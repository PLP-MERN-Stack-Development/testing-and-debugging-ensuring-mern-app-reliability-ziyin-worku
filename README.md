
# MERN Bug Tracker

A full-stack MERN application to report, update, and track bugs. Built with a focus on testing and debugging best practices.

## Features
- Report new bugs via form
- List all bugs
- Update bug status (open → in-progress → resolved)
- Delete bugs
- Robust error handling (backend + frontend)
- Comprehensive tests (unit & integration)

## Tech Stack
- **Backend**: Node.js, Express, Mongoose
- **Frontend**: React (Vite)
- **Testing**: Jest, Supertest, React Testing Library
